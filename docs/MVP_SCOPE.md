# Klio Calendar MVP Scope (v1.0)

## Goal

Launch a working productivity application for Web and Android as quickly as possible.

## Included in v1.0

### Authentication

- Register
- Login
- Logout
- Forgot Password

### Calendar

- Month View Calendar

### Events

- Create Event
- Edit Event
- Delete Event

### Tasks

- Create Task
- Edit Task
- Mark Complete
- Delete Task

### Notes

- Create Note
- Edit Note
- Delete Note

### Reminders

- Event Reminders
- Task Reminders

### Real-Time Sync

- Sync data instantly across devices

## Platforms

- Web (Next.js)
- Android (React Native + Expo)

## Not Included in v1.0

- Search
- Filter
- Labels
- Color Coding
- Countdown Timer
- Recurring Events
- Google Calendar Sync
- Apple Calendar Sync
- Shared Calendars
- Collaboration
- iOS App
- Dark Mode
- Themes
- Analytics

## Success Criteria

A user can:

1. Create an account
2. Login
3. Create events
4. Create tasks
5. Create notes
6. Receive reminders
7. Access the same data on multiple devices
